## 1. CPX Research Integration

### Overview
- **Provider**: CPX Research (cpxresearch.com)
- **Postback URL**: `https://dzcash.com/api/tracking/postback/cpx`
- **Signature**: HMAC-SHA256
- **Click ID Parameter**: `click_id`
- **Offer Sync API**: Available (requires API key)

### Postback Parameters
```
Method: GET
URL: /api/tracking/postback/cpx

Required Query Parameters:
  click_id    - UUID generated by DZCASH
  payout      - Amount network pays DZCASH (e.g., 1.2000)
  status      - Conversion status (1 = complete)
  signature   - HMAC-SHA256 hex string

Signature Calculation:
  HMAC-SHA256(click_id:payout:status, CPX_SECRET)
  
  Example:
  Input: "550e8400-e29b:1.2000:1"
  Key: "my_cpx_secret_key"
  Output: "a1b2c3d4e5f6..."
```

### Signature Validation
```typescript
// backend/src/offers/providers/cpx.provider.ts
async validatePostback(query, headers, body): Promise<boolean> {
  const { click_id, payout, status, signature } = query;
  
  if (!click_id || !payout || !signature) return false;

  const computed = crypto
    .createHmac('sha256', this.secret)
    .update(`${click_id}:${payout}:${status || '1'}`)
    .digest('hex');

  return crypto.timingSafeEqual(Buffer.from(computed), Buffer.from(signature));
}
```

## 2. OfferToro Integration

### Overview
- **Provider**: OfferToro (offertoro.com)
- **Postback URL**: `https://dzcash.com/api/tracking/postback/offertoro`
- **Signature**: MD5
- **Click ID Parameter**: `click_id`
- **Offer Sync API**: Available (requires API key)

### Postback Parameters
```
Method: GET
URL: /api/tracking/postback/offertoro

Required Query Parameters:
  click_id    - UUID generated by DZCASH
  payout      - Amount network pays DZCASH
  o_id        - Offer ID in OfferToro system
  sig         - MD5 hex string

Signature Calculation:
  MD5(o_id:click_id:OFFERTORO_SECRET)
  
  Example:
  Input: "12345:550e8400-e29b:my_offertoro_secret"
  Output: "9f8d7e6c5b4a..."
```

### Signature Validation
```typescript
// backend/src/offers/providers/offertoro.provider.ts
async validatePostback(query, headers, body): Promise<boolean> {
  const { click_id, payout, o_id, sig } = query;
  
  if (!click_id || !payout || !o_id || !sig) return false;

  const hash = crypto.createHash('md5');
  const computed = hash
    .update(`${o_id}:${click_id}:${this.secret}`)
    .digest('hex');

  return sig.toLowerCase() === computed.toLowerCase();
}
```

## 3. Generic Provider

### Overview
- **Provider**: Generic (custom/internal offers)
- **Postback URL**: `https://dzcash.com/api/tracking/postback/generic`
- **Signature**: Pre-shared token
- **Security**: Simple token match (for trusted partners)

### Postback Parameters
```
Method: GET or POST
URL: /api/tracking/postback/generic

Required Query Parameters:
  click_id    - UUID generated by DZCASH
  payout      - Amount network pays DZCASH
  token       - Pre-shared secret token
```

### Token Validation
```typescript
// backend/src/offers/providers/generic.provider.ts
async validatePostback(query, headers, body): Promise<boolean> {
  const { click_id, payout, token } = query;
  
  if (!click_id || !payout || !token) return false;

  return token === this.secretToken;
}
```
